# 🎉 Numerigo – Petualangan Belajar Angka untuk Si Kecil

**Numerigo** adalah aplikasi edukatif berbasis **mobile** yang dirancang khusus untuk anak usia dini (**TK & PAUD**).  
Aplikasi ini membantu anak-anak belajar angka dengan cara **interaktif, menyenangkan, dan penuh imajinasi**.

Dengan pendekatan gamifikasi, Numerigo membuat proses belajar menjadi seperti bermain sehingga anak dapat memahami konsep angka dengan lebih mudah.

Project ini dikembangkan sebagai bagian dari **pengembangan aplikasi edukasi berbasis teknologi** serta sebagai **portfolio project dalam pengembangan aplikasi mobile dan sistem pembelajaran digital**.

---

# 🚀 Features

Berikut fitur utama yang tersedia pada aplikasi **Numerigo**:

### 🎯 Choose It
Mode permainan kuis interaktif di mana anak diminta **memilih jawaban yang tepat** berdasarkan pertanyaan yang muncul di layar.

Tujuan fitur ini adalah untuk membantu anak:
- mengenali angka
- memahami konsep berhitung sederhana
- meningkatkan kemampuan berpikir logis

---

### 🃏 Magic Card
Mode permainan unik yang menggunakan **barcode pada kartu fisik**.

Anak dapat memindai barcode menggunakan kamera pada aplikasi, lalu **jawaban akan muncul secara otomatis di layar**, menciptakan pengalaman belajar seperti **trik sulap yang menyenangkan**.

---

### 📊 Dashboard
Menampilkan **ringkasan aktivitas belajar anak** secara keseluruhan.

Orang tua dapat melihat perkembangan belajar anak secara cepat melalui tampilan dashboard.

---

### 🧒 Child Profile
Fitur untuk membuat **profil anak** sehingga aplikasi dapat dipersonalisasi sesuai dengan pengguna.

Informasi yang dapat disimpan antara lain:
- Nama anak
- Karakter atau avatar
- Riwayat belajar

---

### 📈 Learning Report
Menyediakan **laporan perkembangan belajar anak** yang dapat dipantau oleh orang tua.

Laporan ini membantu orang tua mengetahui:
- progres belajar
- hasil kuis
- perkembangan kemampuan anak

# ⚙️ Installation

Ikuti langkah berikut untuk menjalankan project secara lokal.

### 1. Clone repository

```
git clone https://github.com/Dema08/Bckoffice-Game-Interraktif-Rukun-Harapan-Semester-4.git
```

### 2. Masuk ke folder project

```
cd numerigo
```

### 3. Install dependency

```
flutter pub get
```

### 4. Jalankan aplikasi

```
flutter run
```

Pastikan perangkat emulator atau device sudah terhubung.

# 🎯 Project Goals

Tujuan utama dari pengembangan aplikasi **Numerigo** adalah:
